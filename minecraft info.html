نظرة عامة

هذه الصفحة تشرح كل ما تحتاجه عن الإصدارات الموجودة: Java، Bedrock، والتجمعات اقرأ كل قسم بعناية قبل التحميل والتثبيت.

Java Edition — المتطلبات وطريقة التثبيت

مناسب للاعبين الذين يفضلون التعديلات (mods) وخوادم Java.

المتطلبات الأدني:
  • نظام تشغيل: Windows 7+ / macOS / Linux
  • ذاكرة: 4GB+ (يوصى 8GB للـmods)
  • تثبيت Java Runtime (JRE) 17 أو أحدث
  • مساحة تخزين: 1GB+
خطوات التثبيت:
  1. نزّل ملف الـInstaller من الزر "تحميل مباشر".
  2. شغّل الملف واتبع التعليمات.
  3. إذا أردت تشغيل mods استخدم Launcher يدعمها (مثلاً: Forge أو Fabric).
نصائح أمان:

افحص الملفات بواسطة antivirus وتأكد من مصدر التحميل. لا تشغّل ملفات .jar مجهولة المصدر.

Bedrock Edition — المتطلبات وطريقة التثبيت

نسخة متعددة الأجهزة تعمل على Windows 10/11 والهواتف والأجهزة اللوحية.

المتطلبات:
  • Android
  • مساحة تخزين 1GB+
  • وحدة المعالجة المركزية: بسرعة 800 ميجا هرتز ARM-v7a
  • ذاكرة الوصول العشوائية الرام: 2 جيجابايت.
  • الرسومات: أي نوع مع دعم OpenGL ES 2.0.
  • أندرويد: 4.0.4.
التثبيت:

نزّل الحزمة المناسبة لجهازك واتبع تعليمات التثبيت. على Windows عادةً تكون حزمة UWP من المتجر أفضل لأنها محدثة تلقائياً.

معلومات سريعة

حجم تقريبي

Java: ~200MB — Bedrock: ~300MB

دعم

لو واجهت مشكلة في التثبيت، جرب تشغيل كمسؤول أو راجع المتطلبات من الأعلى.

المشاكل الشائعة وحلول سريعة

  • اللعبة لا تعمل: تأكد من تحديث التعريفات ووجود متطلبات النظام.
  • خطأ في Java: ثبت JRE المناسب (17+).
  • Mods لا تعمل: تأكد من توافق إصدار الـmod مع إصدار اللعبة والـlauncher.
